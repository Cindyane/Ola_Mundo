HML 5

HTML5 é a versão mais recente da linguagem de marcação HTML (Hypertext Markup Language), usada para estruturar o conteúdo de páginas web. Ele traz várias melhorias em relação às versões anteriores, incluindo suporte nativo para vídeos e áudio sem a necessidade de plugins externos, novos elementos semânticos (como article,section, nav) que tornam o código mais organizado e acessível, além de melhorar a interação com APIs para funcionalidades como geolocalização e armazenamento local. Com o HTML5, a criação de sites responsivos e dinâmicos se torna mais fácil e eficiente.

AJavaScript

O Java é uma linguagem de programação orientada a objetos, amplamente utilizada no desenvolvimento de aplicativos para diversos tipos de dispositivos, incluindo desktops, servidores e dispositivos móveis. Criada por James Gosling na década de 1990, a principal característica do Java é a portabilidade, ou seja, o código Java pode ser executado em qualquer plataforma que tenha a Máquina Virtual Java (JVM), sem a necessidade de modificação. Além disso, o Java é muito utilizado em sistemas corporativos, aplicativos Android e desenvolvimento de sistemas em larga escala, devido à sua robustez, segurança e desempenho.
